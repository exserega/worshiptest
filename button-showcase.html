📌 Важные примечания

  • Иконочные кнопки: Используйте 40x40px для основных навигационных кнопок (назад, закрыть, создать)
  • Маленькие иконки: Используйте 32x32px для кнопок в углах карточек (редактировать, удалить)
  • Кнопка удаления: Всегда используйте красный цвет с полупрозрачным фоном
  • Hover эффекты: translateY(-1px) для подъема, scale(1.05) для увеличения
  • Active состояние: scale(0.95) или scale(0.98) для эффекта нажатия
  • Цвета: Всегда используйте CSS переменные, не хардкодьте цвета
  • Иконки: FontAwesome 6, размер 1rem для 40px кнопок, 0.875rem для 32px
  • Transition: all 0.2s ease для плавных анимаций
